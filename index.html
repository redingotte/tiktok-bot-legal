compte2

Official website for the “compte2” application (TikTok Login Kit + Content Posting API integration).

What the app does

compte2 allows its owner to authorize a TikTok account using Login Kit and then publish videos to that same authorized account using the Content Posting API.

We only store access tokens as needed to operate the integration and publish content.

This app is used exclusively by its owner to manage their own TikTok accounts. No third-party users authenticate through this app.

How it works

  1. Login and authorization via TikTok Login Kit (scope: user.info.basic).
  2. Video upload via TikTok Content Posting API (scopes: video.upload / video.publish depending on posting mode).
  3. Publishing the video to the authorized account’s TikTok profile.
