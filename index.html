compte2

Site officiel de l’application “compte2” (intégration TikTok Login Kit + Content Posting API).

À quoi sert l’app

compte2 permet à son propriétaire d’autoriser un compte TikTok via Login Kit, puis de publier des vidéos sur ce même compte via l’API de publication (Content Posting API).

Comment ça fonctionne

  1. Connexion et autorisation via TikTok Login Kit (scope: user.info.basic).
  2. Upload de vidéo via Content Posting API (scopes: video.upload / video.publish selon le mode).
  3. Publication sur le profil du compte autorisé.
